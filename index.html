田口博一｜ゲームプログラマー志望

Unityを用いた個人制作・チーム制作ゲームを掲載しています

自己紹介

ゲームプログラマーを志望している専門学校3年の田口博一です。Unity（C#）を用いた2Dゲームの個人・チーム制作を経験しており、主にアクション性や操作感にこだわった開発をしています。

  • 使用スキル：C# / Unity / Git
  • 制作スタイル：個人制作・チーム制作（最大4人）
  • 開発期間目安：2週間〜1ヶ月

ワッシーのウィッグハント

・1人プレイの横スクロールアクション
・敵を倒してスコアを稼ごう！ 【ジャンル】横スクロールシューティングゲーム
【制作期間】約6カ月（チーム制作）
【使用技術】Unity（C#）
【概要】敵の攻撃をよけながら倒しスコアを稼ごう
【工夫点】何回遊んでも同じパターンにならないように、敵同士に当たり判定をつけたり、かなりランダムな出現に設定しております

泳げマンボウくん

【制作期間】約6カ月（チーム制作）
【使用技術】Unity（C#）

【工夫点】当たり判定の軽量化とリトライ性を意識したテンポ設計を行いました。

その他の制作物

これまでに制作してきたゲームやデータが失われてしまったもの、開発途中で止まってしまった作品、企業様からのプロジェクトなども含め、制作経験の一環として紹介します。

  • 決戦！オレルール！：じゃんけんのカードに属性があり、対決時の攻撃力で体力を削りあう（開発途中で止まってしまった）。
  • Datto!!：兎を操作して3つのレーンを移動し、障害物をよけながら最長距離を移動しよう（完成）。
  • 煩悩退治：お坊さんの中にある煩悩を消して世界を救おう。（完成）
  • ガンバラニャイト：主人公のニャイトのスキルを使い分け、ご主人のチーズを取り返そう（完成）
  • GemHunter：４人対戦のスコアバトル、プレイヤー１人が蛇になり他プレイヤーを妨害する（現在制作中）。
  • ApplevisionProを使用した企業プロジェクト（非公開）：企業様提供のプロジェクトに参加。
